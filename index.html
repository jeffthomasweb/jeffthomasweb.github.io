Buffalo Food Banks

A list and map of some of the food banks located in Buffalo, NY. The goal of this portfolio project is to create a site with Nginx as a reverse proxy, Java Spring as the web framework, Postgres as the database, plot location points on a map using the JavaScript library Leaflet, and have wide test coverage. The site has a JSON API and vanilla JavaScript is used to create HTML from the JSON data. I don't have a domain for the website so unfortunately the website address is an IP address. The website may show as "not secure" in some browsers because I don't have a TLS certificate for https (the site ues http rather than https.)
